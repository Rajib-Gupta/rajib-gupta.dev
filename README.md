## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Rajib-Gupta/portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
